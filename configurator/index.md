# Configurator API

## Overview

The Rivian Configurator has an API for generating images of a particular configuration.

## Base URL
```
https://rivian.com/compimg/
```

## Example Image
```
https://rivian.com/compimg/r1s/2.2/us/exp-lgr_ord-ad3_pkg-lch_whl-1rd@3072x2688.front.webp
```
